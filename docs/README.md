## 프로그래밍 요구사항

- [x] 자바 코드 컨벤션 지키기
- [x] 들여쓰기의 최대 크기는 2
- [x] 3항 연산자 사용하지 않기
- [x] 메소드의 최대 크기는 15
- [x] else 사용하지 않기
- [x] 예외 상황시 에러 문구 [ERROR] 사용하기
